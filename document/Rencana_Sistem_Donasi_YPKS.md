# Perencanaan Pembaruan Sistem YPKS
## Implementasi Modul Program & Sistem Donasi Manual

**Tanggal**: 2 April 2026

### 1. Modul Program Sosial
Digunakan untuk mempublikasikan inisiatif sosial yayasan.
- Katalog program aktif.
- Detail narasi, foto, dan status kegiatan.
- Tombol Donasi di setiap halaman program.

### 2. Sistem Donasi Manual Transfer (Donation Tracking)
Sistem donasi yang aman, efisien, dan tanpa biaya aplikasi.

#### Alur Kerja Donatur:
1. Pilih program & isi formulir donasi.
2. Dapatkan **Ticket ID** unik (misal: `YPKS-24-0012`).
3. Lakukan transfer manual ke rekening resmi yayasan.
4. Unggah foto bukti transfer di halaman konfirmasi.
5. Cek status donasi (Pending/Verified) menggunakan Ticket ID.

#### Keunggulan:
- **Zero Fees**: Tidak ada potongan biaya dari Payment Gateway (MDR 0%).
- **Keamanan**: Dana langsung ke rekening yayasan.
- **Transparansi**: Donatur mendapatkan nomor referensi resmi.

### 3. Dashboard Admin
Admin dapat memproses konfirmasi donasi secara manual melalui panel backend yang terintegrasi.
